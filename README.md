## 📌 About

This repository consolidates solutions, notes, and practice problems across multiple competitive programming platforms. Each solution is organized by platform and problem category, with a focus on clarity and learning progression.

- **SBC** – São Paulo Programming Marathon (Maratona de Programação)

---

## 🎯 Goals

- Strengthen problem-solving and algorithmic thinking
- Study core data structures: arrays, maps, sets, graphs, trees
- Prepare for olympiad-level contests (OBI, Maratona SBC)
- Build a consistent practice habit and track progress over time

---

## 📈 Progress

Problems are being added progressively. Difficulty ranges from beginner to advanced, covering topics such as sorting, searching, dynamic programming, and graph algorithms.
